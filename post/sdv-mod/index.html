4. 增量 Mod 安装的方法

一些汉化 mod 或者修改其他 mod 的 mod 并不是完整的独立 mod，可能只包含国际化文件等，需要和源 mod 合并，或者替换源 mod 中的同名文件，有两种方法：

  1. 先将源 mod 解压后替换文件再打包，使用启动器安装。
  2. 先安装源 mod，mod 安装位置在 /Android/data/abc.smapi.gameloader/files/Mods，安卓上一般的文件管理器没有写入 /Andriod/data 的权限，可以连接电脑，在电脑上替换文件（可能需要启用 usb 调试）。或者使用支持 shizuku 的文件管理器，可以直接管理/Android/data 目录
